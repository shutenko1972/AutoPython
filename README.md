# Автоматизированные тесты для сайта Saucedemo

Этот проект содержит набор автоматизированных тестов для веб-приложения [Saucedemo](https://www.saucedemo.com/), разработанных с использованием Python и Selenium WebDriver.

## Тестовые сценарии

Реализованы следующие сценарии тестирования:

1. **Успешный вход в систему** (standard_user / secret_sauce)
2. **Вход с неверным паролем**
3. **Вход заблокированного пользователя** (locked_out_user)
4. **Вход с пустыми полями**
5. **Вход пользователем с проблемами производительности** (performance_glitch_user)

## Системные требования

### Для локального запуска
- Python 3.10 или выше
- Браузер Google Chrome
- Git (опционально, для клонирования репозитория)

## Быстрый старт (Windows)

1. **Скопируйте всю папку проекта** на ваш компьютер
2. **Установите зависимости**, запустив файл `install.bat`
3. **Запустите тесты**, выполнив файл `run_simple.bat`

## Подробные инструкции по настройке

### Использование терминала/командной строки

1. **Создайте виртуальное окружение:**
   ```bash
   python -m venv venv
   ```

2. **Активируйте виртуальное окружение:**
   - **Windows:**
     ```bash
     venv\Scripts\activate
     ```
   - **macOS/Linux:**
     ```bash
     source venv/bin/activate
     ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите тесты:**
   ```bash
   pytest tests/ -v
   ```

### Использование Docker

1. **Соберите Docker-образ:**
   ```bash
   docker build -t saucedemo-tests .
   ```

2. **Запустите контейнер:**
   ```bash
   docker run --shm-size=2g saucedemo-tests
   ```
